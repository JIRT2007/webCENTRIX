Manual

Presentation

  • CentrixOS is a customization layer for GNU/Linux designed to provide a preconfigured Hyprland desktop environment.

  • This project is developed by JIRT2007 and follows a principle of minimalism that seeks to install the minimum and necessary to offer an aesthetic and correctly preconfigured desktop environment.

  • The project is inspired by Omarchy, developed by David Heinemeier Hansson (DHH).

Keybindings

  • F1 (Launch terminal)

  • F2 (Close program)

  • F3 (Launch Wofi)

  • Alt + Q (Floating window)

  • Ctrl + Alt + <-- (Move to the workspace on the left)

  • Ctrl + Alt + --> (Move to the workspace on the right)

  • Alt + Click (Move window)

  • Alt + Shift + Number (Move window to the specified workspace)

  • Insert (Screenshot)

Centrix Control

  • Centrix Control is a program developed by JIRT2007 using Shell Script to manage simple CentrixOS functions.

  • It is executed using the 'ctrix' command and is developed for systems x86_64 with SystemD.

  • The program allows you to manage system customizations, simple control of virtual machines with QEMU and offers a menu for equipment energy management.
