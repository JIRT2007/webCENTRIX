Manual

Presentation

  • CentrixCL is a customization layer for GNU/Linux designed to provide a preconfigured Hyprland desktop environment.

  • This project is developed by JIRT2007 and follows a principle of minimalism that seeks to install the minimum and necessary to offer an aesthetic and correctly preconfigured desktop environment.

  • The project is inspired by Omarchy, developed by David Heinemeier Hansson (DHH).

Installation steps

  • sudo pacman -S git

  • git config --global user.name "USERNAME"

  • git config --global user.email "USERMAIL"

  • git clone https://github.com/JIRT2007/CentrixCL.git

  • chmod +x CentrixCL/install.sh

  • ./CentrixCL/install.sh

  • 1) Install CentrixCL.

  • 0) Exit.

  • sudo reboot now

WARNING

  • CentrixCL can contain bugs like any software product.

  • It is recommended to install CentrixCL on a "raw" operating system and preferably in TTY mode.

Keybindings

  • Alt + Q (Launch terminal)

  • Alt + C (Close program)

  • Alt + W (Launch Fuzzel)

  • Alt + V (Floating window)

  • Alt + Click (Move window)

  • Alt + Shift + Number (Move window to the specified workspace)

CentrixCL Control

  • CentrixCL Control is a program developed by JIRT2007 using Shell Script to manage simple CentrixCL functions.

  • It is executed using the 'cmenu' command and is developed for systems x86_64 with SystemD.

  • The program allows you to manage system customizations, simple control of virtual machines with QEMU and offers a menu for equipment energy management.
